Hi, I'm Hugo Feliciano.

I'm a graduate Game Developer from Lisbon, Portugal.
I work on projects as a Programmer, mainly using C# / Python.
I have also worked on projects as a Game Designer, Animator, Texture Artist and UI Artist.






Projects

Here is a showcase of all the projects I've worked on so far.

The Sixth Sun

Retro action First-Person Shooter game set during the peak of the Aztec Civilization, where the player is an aztec warrior chosen by the gods to fight an alien invasion.


This game was developed with Unity 3D.
I mainly worked as a Programmer, developing the combat systems and mechanics, and implementing half of the enemies and their behaviours.
I also worked as an Animator and Artist for the game, being responsible for animating and texturing every enemy and object in the game, as well as designing and implementing the UI.

SEE THE GAME ON ITCH.IO

Trinity

Fast-paced local multiplayer third-person shooter that employs the rock-paper-scissors logic to the player's projectiles. It features several game modes and procedurally generated arenas.


This game was developed with Unity 3D.
I mainly worked as a Programmer, developing the players' movement system and implementing all game modes, as well as a tool that allows the Game Designer to create endless new game modes in the editor. I also developed a Leaderboard system that saves the top performances in every game mode.
In addition, I was responsible for designing and implementing the UI.

Jumper

2D Platformer game made in pixel art, where the player has to infiltrate an enemy base, using it's attack abilities and a teleport mechanic inspired by the movie Jumper (2008).


This game was developed with Unity 2D.
I worked as the only Programmer, developing every system and mechanic in the game, like the movement and combat systems, and the teleport mechanic.
I also implemented every animation made by other members of the team for the characters and objects in the game.

Download the game

Space Conqueror

2D Side-Scrolling Endless Runner game, where the player controls a spaceship by moving it up/down and shooting lasers at passing enemy spaceships.


This game was developed with Unity 2D
It is a joint project with CICANT, with the purpose of creating an inclusive and accessible game for people with physical and mental limitations, providing them with a stimulating and fun experience.
I mainly worked as the only Programmer, developing every system and mechanic implemented in the game.
I also worked as a Game Designer, balancing the entire game experience and making sure the game is playable for people with physical and mental limitations.

Hooked

2D Endless Fishing game, where the player uses the mouse to control the fish hook and catch as many fish as he can while avoiding obstacles.


This game was developed with Unity 2D.
It is a joint project with CICANT, and was made in the same context as Space Conqueror.
I mainly worked as the only Programmer, developing every system and mechanic implemented in the game.
I also worked on the Game Design, having balanced the game mechanics and the game's overall difficulty to make it accessible for people with physical and mental limitations.
